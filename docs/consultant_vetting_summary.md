# 移民顾问审核规范总结

## 顾问分级系统

### A类：持牌顾问/律师
- **资质要求**：加拿大 RCIC 执照或律师执照
- **服务范围**：全流程移民/签证服务
- **审核要求**：
  - RCIC 执照编号
  - 执业机构信息
  - 可公开查询链接
  - 定期复核（6-12个月）

### B类：留学/签证顾问
- **资质要求**：有行业经验但非移民持牌
- **服务范围**：留学/访签/材料辅助
- **审核要求**：
  - 从业年限说明
  - 过往案例描述
  - 服务范围声明
  - 明确标注"非持牌移民顾问"

### C类：文案/辅助人员
- **资质要求**：文书、翻译、材料整理
- **服务范围**：不可独立接单
- **限制**：
  - 不展示在搜索结果首页
  - 不允许独立报价
  - 只能作为协作人员

## 入驻审核流程

### Step 1：基础身份审核（强制）
- 真实姓名（中/英）
- 身份证件或护照
- 联系方式（邮箱+电话）
- 居住国家/城市
- 真人视频认证

### Step 2：执照/资质审核（分级）
- A类：执照验证+定期复核
- B类：从业经验审核
- C类：协作资格审核

## 注册流程设计

### 顾问注册步骤：
1. 邮箱验证（首次注册）
2. 基础信息填写
3. 选择顾问类型（A/B/C）
4. 上传资质证明
5. 提交审核
6. 等待平台审核
7. 审核通过后激活账户

### 审核状态：
- `pending` - 待审核
- `under_review` - 审核中
- `approved` - 已通过
- `rejected` - 已拒绝
- `suspended` - 已暂停

## 数据库字段需求

### RCIC 表需要的字段：
- `consultantType` - 顾问类型（A/B/C）
- `licenseNumber` - 执照编号（A类必填）
- `licenseVerified` - 执照是否验证
- `licenseExpiryDate` - 执照过期日期
- `yearsOfExperience` - 从业年限
- `approvalStatus` - 审核状态
- `approvalNotes` - 审核备注
- `documentsSubmitted` - 提交的文档列表
- `videoVerified` - 视频认证状态
- `lastReviewDate` - 最后审核日期
- `nextReviewDate` - 下次审核日期

## 禁止行为（零容忍）
1. 承诺 100% 获批
2. 伪造材料/教唆造假
3. 私下交易绕过平台
4. 虚构成功案例
5. 隐瞒顾问真实身份或资质
6. 超范围执业

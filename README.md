# MapleBridge Workspace

This repository is a product workspace for MapleBridge-related experiments and application code. It is not the main public open-source entry point.

## Start Here

- MapleBridge product: https://maplebridge.io/
- MapleBridge app: https://maplebridge.io/app
- MapleBridge Open docs: https://maplebridge.io/open/
- Main open-source repository: https://github.com/jinjihuang88-ui/maplebridge-open

## What To Promote

Promote MapleBridge when the audience is a buyer, supplier, ecommerce seller, sourcing operator, importer, or founder trying to understand China sourcing.

Plain version:

> MapleBridge helps North American buyers turn unclear sourcing requests into structured supplier-matching conversations with Chinese manufacturers.

Promote MapleBridge Open when the audience is technical: developers, procurement-tool builders, AI-agent builders, open-source readers, or potential integration partners.

Plain version:

> MapleBridge Open shows the public schema, examples, and demo behind buyer intent, supplier capability, match explanations, and human review handoffs.

## Repository Boundary

Use this repository as a private or internal workspace unless it has been cleaned for public readers.

Do not promote this repository as the public GitHub entry point while it contains mixed experiments, deployment notes, private workspace files, or configuration history.

For public examples, issues, and contribution tasks, point people to:

https://github.com/jinjihuang88-ui/maplebridge-open

## Security Note

Do not commit secrets, live credentials, customer records, supplier private data, crawler sources, ranking thresholds, or production-only operational details.

If this repository remains public, review tracked environment files and rotate any exposed credentials before using it in promotion.
